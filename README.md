# مشروع تطوع65x - منصة التطوع الرقمي

## نظرة عامة على المشروع

مشروع "تطوع65x" هو موقع إلكتروني متكامل للتطوع، يهدف إلى ربط المتطوعين مع الفرص التطوعية المتاحة في مختلف أنحاء المملكة العربية السعودية. تم تطوير الموقع باستخدام تقنيات حديثة توفر تجربة مستخدم سلسة وممتعة.

## التقنيات المستخدمة

### التقنيات الرئيسية
- **React 19**: مكتبة JavaScript الحديثة لبناء واجهات المستخدم التفاعلية
- **Vite**: أداة بناء سريعة وحديثة للشاريع الويب
- **React Router**: لإدارة التوجيه والتنقل بين الصفحات
- **CSS3**: لتنسيق الموقع بتصميم عصري وجذاب

### الإضافات والمكتبات
- نظام Flexbox و Grid للتصميم المتجاوب
- تقنية Backdrop Filter للتأثيرات الزجاجية
- أيقونات SVG للتوضيح
- تصميم متجاوب يدعم جميع أحجام الشاشات

## صفحات الموقع

### 1. الصفحة الرئيسية (/)
- قسم Hero كبير مع زر دعوة للعمل
- عرض ميزات الموقع بشكل جذاب
- تصميم تفاعلي مع تأثيرات حركية

### 2. صفحة البحث (/search)
- صندوق بحث متقدم
- فلات متعددة (الموقع، النوع، التاريخ)
- عرض نتائج البحث بشكل بطاقات جذابة
- تصميم متجاوب للأجهزة المختلفة

### 3. صفحة الفعاليات (/calendar)
- عرض الفعاليات القادمة
- تقويم شهري مع إمكانية التنقل
- فلات حسب نوع الفعالية (بيئي، اجتماعي، تعليمي، صحي)
- بطاقات فعالية تفاعلية

### 4. صفحة من نحن (/about)
- عرض رسالة المشروع وأهدافه
- قسم القيم والمبادئ
- معلومات عن الفريق

### 5. صفحة اتصل بنا (/contact)
- نموذج اتصال مع التحقق من البيانات
- عرض معلومات التواصل
- رسائل نجاح/خطأ تفاعلية

### 6. صفحة تسجيل الدخول (/login)
- نموذج تسجيل الدخول
- رابط للتسجيل
- تصميم أنيق وآمن

### 7. صفحة إنشاء حساب (/signup)
- نموذج التسجيل مع التحقق الكامل
- التحقق من: الاسم، البريد الإلكتروني، كلمة المرور
- رسائل خطأ واضحة

### 8. صفحة 404 (غير موجودة)
- صفحة خطأ مخصصة
- زر العودة للرئيسية
- تصميم جذاب وهادف

## المزايا الرئيسية

### التصميم المتجاوب
- دعم جميع أحجام الشاشات (موبايل، تابلت، كمبيوتر)
-_detection تلقائي نوع الجهاز
- قائمة تنقل خاصة بالموبايل مع زر الهامبرجر
- أزرار تسجيل دخول وتسجيل مصغرة للموبايل

### التحقق من النماذج
- التحقق من صحة جميع الحقول
- رسائل خطأ واضحة ومفيدة
- تأثيرات بصرية للحالات المختلفة
- دعم اللغة العربية (RTL)

### تجربة المستخدم
- خلفية شفافة تظهر صورة الخلفية
- تأثيرات زجاجية حديثة
- انتقالات سلسة وحركية
- زر العودة للأعلى
- تحميل ديناميكي للمحتوى

### الأصول والرسومات
- شعارات بصيغة SVG عالية الجودة
- خلفيات متوافقة مع جميع الشاشات
- أيقونات تفاعلية

## كيفية التشغيل

